Samen naar Budapest - versie 4.0 preview

Upload naar GitHub:
1. Pak de zip uit.
2. Upload de inhoud van deze map naar de hoofdmap van je repository.
3. Zorg dat index.html, stijl.css, app.js en de map assets rechtstreeks in de hoofdmap staan.
4. Commit changes.
5. Wacht tot GitHub Pages groen is.
6. Open de website en druk Ctrl + F5.

Persoonlijke teksten kunnen later in app.js worden vervangen bij profiles.
